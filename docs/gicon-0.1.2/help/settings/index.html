Toolbar

The information displayed in the Settings window, is controlled via this toolbar. The buttons have the following behavior:

  • Download - will update all the displayed information, by reading the cameras current status. This might take a while, especially on 1200bps. As the Settings window is displayed the the application will automatically download the camera settings. These settings will become the default camera settings.
  • Upload - will upload the changed settings into the camera.
  • Default - will restore the default camera settings, stored when the Settings window was displayed.
  • Help - displays this page.


Basic Settings

There are four basic settings allowing you to customize the behavior of your camera.

  • Command Dial - allows you to select between Normal, and Reverse indicating the active direction of the command dial.
  • Long Exposure - allows you to select between Bulb (B), and Time (T) options. When the Bulb mode is selected, the shutter will stay open until the shutter release button is depressed, and closed when it is released. The Time mode will keep the shutter open after the shutter release button is depressed, and stay opened until the shutter release button is depressed for a second time.
  • ISO Priority - in Manual mode you can override the DX coded film speed, by changing the ISO setting. If the DX option is selected and a DX coded film is loaded into the camera, no ISO setting change is possible.
  • Dual Release - when the camera is interconnected with another camera, this feature will allow you to synchronize the shutter release of the cameras. Possible options are: Independent, Alternative, Simultaneous and Disabled.


Memo Holder

The camera can store the shooting data about each frame you take. By default, it stores no data. These controls will allows you to enable and customize the feature. However there are two limitations:

  • the feature can be enabled only if there is no film in the camera.
  • the format of the data can be changed only if there is no shooting data stored in the camera's memo holder.

The memo holder's size is limited. If the memo holder fills, the camera offers two possibilities to handle the situation. By default the camera will Stop Storing Memo Holder Data When Full. The second option will Force Download When Memo Holder is Full by preventing you from taking any further pictures. If this happens but you cannot download the data, it is possible to override the feature by turning off then back on the camera.


Viewfinder

Two viewfinder options are available. The first will Display the Frame Counter in Program Modes, replacing the Vari Program Mode Indicator.
The second option in Program and Aperture Priority Modes will display the difference between the exposure metered in Matrix and the Center Weighted metering modes. A positive value indicates that the Matrix Metering is overexposing, a negative value indicates that it is underexposing relative to the Center Weighted Metering.


Auto Focus and Film Transport

In the Continuos Shooting Mode, pressing the shutter release will fire the camera even if the subject is out of focus. By enabling Focus-Release priority in Continuos Mode the camera will first focus on the subject and then fire the shutter. This way the first frame will be in focus, but the camera will not refocus between frames. By enabling Focus Between Frames in Continuos Mode, you can make sure that all frames taken are in focus.
In Single Shooting Mode, by default, the camera will not fire the shutter unless the subject is in focus. You can turn off this feature by enabling Release-Focus Priority in Single Mode.
The Simultaneous AE-AF Lock and Trap Focusing are not yet working, therefore they are disabled.


Beeper

This feature is perhaps the most useless among the custom settings. By default the camera beeper is disabled, however you can customize it, so that the camera will beep:

  • On Film Error - no film, end of film, jam
  • On Exposure Error - whenever there is an exposure error displayed on the LCD.
  • During Self Timer - as self timer counts down, the camera beeps as the Self Timer Indicator LED blinks.
  • When in Focus - beeps twice when in focus.


Miscellaneous

If your camera is equipped with an MF-26 Control Back you can enable the Print of the Roll Number on the beginning of the Film.
The second option, will enable Easy Compensation In Aperture Mode. In Aperture Priority Mode you will be able to set the compensation by the command dial, and see in the viewfinder the exposure compensation. This setting will override the Display Matrix - Center Meter Delta viewfinder option.
The Easy Compensation will be added to the compensation set via the exposure compensation button.
