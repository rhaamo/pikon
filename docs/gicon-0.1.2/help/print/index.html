Printer Controls

Postscript printer support is built-in gIcon. You can chose via Print To whether the printed stream is directed into a File or a Printer. If Printer is selected you can specify and customize the printer through the Printer Command entry. If File is selected, you must specify a file to print to. The file will be in postscript format. Some default printer controls are also available, like Orientation, and Paper Size.


Print Layout

These controls allows you to set your print layout. The Columns field will indicate how many vertical columns will be printed. The Print Caption and Print Title checkboxes will enable the printing of the caption and the title on each of the labels. You can set the label's dimension, and the space between the labels in horizontal and vertical directions, as well the offset of the first
row/column of labels from the margins of the sheet. The Copyright note will be attached to each label.


The Label

This is the result. The copyright note is customizable as described above. The time stamp looks awkward but contains all the necessary information about the time the data was downloaded.

If any of the customizable texts (copyright notes, roll title, caption) are too long, they will not be formatted to the size of the label. It is wise to print into a file first, check the result and only when it looks fine, print on labels.
