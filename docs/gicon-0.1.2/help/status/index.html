Toolbar

The information displayed in the Status window, is controlled via this toolbar. The buttons has the following behavior:

  • Download - will update the displayed information, by reading the camera's current status. This will parse the memo holder, therefore it will take as much time as a download would take.
  • Help - displays this page.


Memory Status

The memory status displays the Free, Used and Total Bytes of the memory holding the shooting data.


Roll Status

The Stored roll number is a positive integer and represents the number of recorded rolls, ready for download. The + sign means that a roll's shooting data is currently stored in the camera but it is not yet available for download.

The Free Bytes, displayed in the memory status is translated into rolls. There are three roll formats, accordingly there are three estimations for each of these formats. Note that format change is possible only as there is no data stored in the camera.
