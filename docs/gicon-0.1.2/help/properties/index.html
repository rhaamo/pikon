Communication

At the moment. gIcon supports only the Nikon F90X camera, but there are plans to extend its capabilities to be able to handle other cameras as well. Therefore the Camera control is permanently disabled. You can chose the Device, which will connect your camera and host. This might be one of four serial ports or if you have a special serial port you can specify it in the other entry.
Two serial Speeds are supported, the recommend and default speed is 9600 bps, as it provides higher performance.


Default formats

When saving a file, gIcon will attempt to determine the desired file format by the extension. If no extension is suppied gIcon will use the Default File Format specified in the Preferences dialog.
